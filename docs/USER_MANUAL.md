## 快速开始

### 注册账号

1. 访问系统首页
2. 点击右上角「注册」按钮
3. 填写用户名、邮箱、密码
4. 阅读并同意用户协议和隐私政策
5. 点击「注册」完成账号创建

### 登录系统

1. 访问系统首页
2. 点击右上角「登录」按钮
3. 输入用户名/邮箱和密码
4. 点击「登录」进入系统

### 首次使用

登录后，系统会引导您完成以下设置：
1. **选择目标考试**（可选）：帮助系统推荐相关内容
2. **设置个人偏好**：主题、字体大小等
3. **完成新手引导**：了解主要功能

## 功能介绍

### 1. 错题录入

将做错的题目录入系统，建立个人错题库。

#### 支持的题型

| 题型 | 图标 | 说明 |
|------|------|------|
| 单选题 | A | 从多个选项中选择一个正确答案 |
| 多选题 | ☑ | 从多个选项中选择多个正确答案 |
| 判断题 | ✓ | 判断陈述正确与否 |
| 填空题 | ___ | 填入空白处的答案 |
| 简答题 | 📝 | 简要回答问题 |
| 计算题 | 🔢 | 需要计算过程的题目 |
| 其他 | • | 其他类型题目 |

#### 录入步骤

1. **选择科目**：点击科目图标选择对应科目
2. **输入内容**：在编辑框中粘贴题目
3. **查看解析**：系统自动解析题目内容
4. **确认保存**：确认无误后点击保存

#### 智能解析规则

系统会自动识别以下内容：
- **题目文本**：题目描述
- **选项**：A、B、C、D 等选项
- **答案**：正确答案
- **解析**：答案解析

### 2. 智能组卷

根据您的需求自动生成练习试卷。

#### 筛选条件

- **知识点**：按知识点范围筛选
- **题型**：选择题目类型
- **难度**：简单/中等/困难
- **状态**：未掌握/学习中/已掌握
- **数量**：设置题目数量

#### 组卷模式

| 模式 | 说明 |
|------|------|
| 随机模式 | 从错题本随机抽取 |
| 薄弱模式 | 优先选择错误率高的题目 |
| 收藏模式 | 仅包含收藏的题目 |
| 指定模式 | 按指定条件筛选 |

### 3. 在线答题

在线完成练习，系统自动计时和评分。

#### 答题界面

```
┌─────────────────────────────────────┐
│  [题目内容]                          │
│                                     │
│  A. 选项一                          │
│  B. 选项二                          │
│  C. 选项三                          │
│  D. 选项四                          │
│                                     │
│  [上一题]  [提交答案]  [下一题]      │
└─────────────────────────────────────┘
```

#### 答题工具

- **答题卡**：查看答题进度，快速跳转
- **画笔**：在题目上勾画标注
- **计算器**：快速计算
- **草稿**：临时记录思路
- **收藏**：标记重要题目

### 4. 复习系统

基于间隔重复算法，科学安排复习计划。

#### Leitner 复习法

```
┌─────────────────────────────────────┐
│  箱子1（每天复习）    │  →  新题目
│  箱子2（每3天复习）   │  →  掌握中
│  箱子3（每周复习）    │  →  基本掌握
│  箱子4（每两周复习）  │  →  熟练
│  箱子5（每月复习）    │  →  精通
└─────────────────────────────────────┘
```

#### 复习反馈

每次复习后，根据您的掌握程度选择：
- **Again**：不记得，需要重新学习
- **Hard**：有些困难，需要更多练习
- **Good**：基本掌握，按计划复习
- **Easy**：完全掌握，延后复习

### 5. 学习统计

可视化展示学习数据和进步趋势。

#### 统计指标

- **总题数**：录入的错题总数
- **正确数**：答对的题目数
- **错误数**：答错的题目数
- **正确率**：答题准确度
- **学习时长**：累计学习时间
- **学习天数**：活跃学习天数
- **日均学习**：平均每日学习时间
- **学习效率**：单位时间掌握题目数

#### 时间范围

- 今日
- 本周
- 本月
- 本季度
- 本年度
- 全部

### 6. 个人中心

管理个人信息和系统设置。

#### 个人资料

- 修改头像
- 更新昵称
- 更改密码
- 绑定邮箱

#### 系统设置

| 设置项 | 选项 |
|--------|------|
| 主题 | 浅色 / 深色 / 自动 |
| 字体大小 | 小 / 中 / 大 |
| 自动跳转 | 开启 / 关闭 |
| 通知提醒 | 开启 / 关闭 |

## 使用指南

### 典型使用流程

```
1. 录入错题 → 选择科目 → 粘贴题目 → 确认解析 → 保存
                    ↓
2. 智能组卷 → 设置筛选条件 → 生成试卷 → 开始练习
                    ↓
3. 在线答题 → 选择答案 → 使用工具辅助 → 提交答案
                    ↓
4. 查看结果 → 查看正确率 → 错题解析 → 加入错题本
                    ↓
5. 定期复习 → 查看待复习 → 完成复习 → 反馈掌握程度
                    ↓
6. 查看统计 → 分析数据 → 调整学习计划
```

### 高效学习建议

1. **及时录入**：做错题目后立即录入，保持新鲜度
2. **详细解析**：录入时添加详细的解析，便于后续复习
3. **定期复习**：遵循系统推荐的复习计划，及时复习
4. **专注薄弱**：优先复习标记为"Again"和"Hard"的题目
5. **分析统计**：定期查看学习统计，了解进步趋势
6. **合理规划**：每日安排固定时间进行错题复习

## 常见问题

### Q1：题目解析不准确怎么办？

A：系统使用智能算法解析题目，但复杂题目可能需要手动调整。您可以：
1. 检查输入格式是否符合规范
2. 手动编辑解析结果
3. 联系技术支持

### Q2：如何批量导入错题？

A：目前系统支持单个录入，批量导入功能正在开发中。

### Q3：复习计划可以自定义吗？

A：系统基于 Leitner 算法自动生成复习计划，您可以通过反馈掌握程度调整下次复习时间。

### Q4：数据会丢失吗？

A：所有数据存储在云端，支持自动备份，不会丢失。

### Q5：如何导出学习数据？

A：在「学习统计」页面点击「导出」按钮，可选择导出为 PDF 或 Excel 格式。

### Q6：可以离线使用吗？

A：目前系统需要联网使用，离线功能正在开发中。

### Q7：如何分享错题给同学？

A：暂不支持直接分享，您可以截图分享或导出后发送。

### Q8：忘记密码怎么办？

A：点击登录页面的「忘记密码」，通过注册邮箱重置密码。
