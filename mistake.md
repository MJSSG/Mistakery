## 发现的问题

### 1. 错题详情页面加载异常 ✅ 已修复
- **优先级**: 高
- **模块**: 错题管理
- **描述**: 在错题列表页面点击错题卡片时，页面跳转到详情页但内容无法正确加载
- **复现步骤**:
  1. 登录系统
  2. 进入错题列表页面
  3. 点击错题卡片上的菜单按钮
  4. 页面跳转到 /mistake/:id 详情页
  5. 页面只显示头部导航，错题详情内容区域为空
- **影响**: 用户无法查看错题详情、无法编辑错题
- **修复方案**:
  - 创建 `mistake.service.ts` 服务文件，实现错题相关 API 调用
  - 修复 `MistakeDetailView.vue` 中的 `loadMistake` 函数，启用 API 调用
  - 提交：b1f03fc
- **修复时间**: 2026年2月14日

### 2. 删除操作后前端显示错误
- **优先级**: 中
- **模块**: 错题管理
- **描述**: 批量删除错题后，前端出现 JavaScript 错误
- **错误信息**: `mistakes.value.filter is not a function`
- **复现步骤**:
  1. 登录系统
  2. 进入错题列表页面
  3. 勾选错题复选框
  4. 点击"删除"按钮
  5. 在确认对话框点击"OK"
  6. 错题被成功删除，但前端控制台报错
- **影响**: 删除功能本身正常工作，但前端存在代码错误

### 3. 错题录入科目选择 Bug
- **优先级**: 高
- **模块**: 错题录入
- **描述**: 错题录入时选择科目后，subjectId 未正确传递到后端
- **错误信息**: `Bad Request Exception: subjectId should not be empty`
- **复现步骤**:
  1. 登录系统
  2. 进入错题录入页面
  3. 选择科目（如：政治理论）
  4. 填写题目内容、答案、解析
  5. 点击"确认录入"按钮
  6. 后端返回 400 错误：subjectId should not be empty
- **影响**: 错题录入功能无法正常使用
